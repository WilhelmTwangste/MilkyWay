## О проекте
**MilkyWay** — мобильное приложение-платформа, разработанное в рамках выпускной квалификационной работы. Проект представляет собой образовательную среду для развития когнитивных функций пользователей различных возрастов через интерактивные визуальные новеллы и мини-игры.

Ключевая концепция основана на нейропсихологических исследованиях Карла Прибрама («Языки мозга»), подтверждающих, что эмоциональное вовлечение через геймификацию эффективно активирует нейронные сети, ответственные за память, внимание, логическое мышление и принятие решений.

---

## Основные возможности

### Персонализация и безопасность
- Аутентификация через Firebase (email, пароль).  
- Возрастная адаптация: контент автоматически подстраивается под пользователей младше и старше 18 лет.  
- Безопасное хранение данных об аккаунтах (Firebase Realtime Database).  

### Интерактивные визуальные новеллы
Сюжетные истории с ветвлениями сюжета на основе выбора игрока.  
Тематика: история, наука, математика, физика, астрономия.  
Анализ решений по завершении новеллы — статистика и рефлексия.  
Эмоциональное погружение через качественную 2D/3D-графику.  

### Мини-игры для тренировки когнитивных функций

**Sudoku:**  
Три уровня сложности  
Автоматическая генерация корректных головоломок  
Проверка решений, подсветка ошибок

**DoodleJump:**  
Бесконечная генерация платформ  
Управление через акселерометр или клавиатуру  
Подсчет очков и сохранение рекордов

## Технические особенности
Кроссплатформенность: Android 8.0+, iOS 12.0+  
Архитектура: MVC (Model-View-Controller) для модульности и масштабируемости  
Адаптивный интерфейс с Modern UI Pack  
3D-графика, созданная в Blender, ZBrush и Daz Studio

## Технологический стек

| Компонент | Технология / Инструмент | Назначение |
|-----------|------------------------|------------|
| Игровой движок | Unity 2022.3+ | Разработка кроссплатформенного приложения с 2D графикой |
| Язык программирования | C# 8.0+ | Основная логика, объектно-ориентированное проектирование |
| Бэкенд и БД | Firebase Authentication, Realtime Database | Аутентификация, хранение профилей |
| Графика и 3D | Blender, ZBrush, Daz Studio | Создание 3D-моделей персонажей, ландшафтов и элементов интерфейса |
| UI/UX | Modern UI Pack, TextMesh Pro | Современный, адаптивный и интуитивный пользовательский интерфейс |

## Структура проекта (Assets)

| Папка / Файл | Содержание |
|--------------|------------|
| **`Scenes/`** | Сцены Unity для разных частей приложения |
| `Registration.unity` | Сцена регистрации |
| `MainMenu.unity` | Главное меню |
| `Sudoku.unity` | Сцена Sudoku |
| `DoodleJump.unity` | Сцена DoodleJump |
| `Novel.unity` | Сцена визуальной новеллы |
| **`Scripts/`** | Основные скрипты на C# |
| `RegistrationManager.cs` | Логика регистрации и аутентификации |
| `NovelManager.cs` | Управление визуальными новеллами |
| `SudokuManager.cs` | Логика игры Sudoku |
| `DoodleJumpPlayer.cs` | Логика игрока в DoodleJump |
| `ContentManager.cs` | Управление контентом и данными |
| `...` | Другие скрипты по функционалу |
| **`Prefabs/`** | Переиспользуемые игровые объекты |
| `Player.prefab` | Игрок для DoodleJump |
| `Platform.prefab` | Платформы для игр |
| `UIElements.prefab` | Элементы интерфейса |
| **`Resources/`** | Динамически загружаемые данные |
| `Dialogs/` | Диалоги для новелл |
| `Configs/` | Конфигурационные файлы |
| **`Images/`** | 2D-графика, спрайты, текстуры |
| **`Modern UI Pack/`** | Готовые UI-компоненты |
| **`TextMesh Pro/`** | Библиотека для работы с текстом |
| **`Plugins/`** | Firebase SDK и сторонние плагины |
| **`Kenney/`** | Ассеты для графики (по необходимости) |
| **`Editor/`** | Скрипты для редактора Unity |
